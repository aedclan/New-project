# Personal Hub 当前功能总结

更新时间：2026-06-27

当前项目已经从“本地个人工具”发展成一个可部署到 VPS 的个人 / 家庭工作台。它不是单纯的记账网页，而是一个包含账号、家庭收支、订阅、人情往来、事项、笔记、收藏、数据同步、备份部署的综合型个人数据系统。

## 一、账号与权限

- 支持服务器账号登录。
- 支持账号注册。
- 支持管理员与普通用户区分。
- 普通用户只能访问自己的数据。
- 管理员可以管理用户。
- 登录状态使用服务器 Session / Cookie 保存。
- 用户数据按账号隔离保存。

## 二、数据同步

- 登录后读取服务器数据。
- 修改数据后可同步到服务器。
- 支持同账号多端使用的基础同步能力。
- 支持服务器数据状态检查。
- 支持从服务器恢复数据。
- 支持完整 JSON 备份导出 / 导入。

## 三、生活收支

- 支持生活收支记录。
- 支持收入、支出分类。
- 支持支付宝账单导入。
- 支持微信账单导入规划与部分识别能力。
- 导入时可选择归属：男方、女方、共同。
- 支持历史月份查看。
- 支持月度数据与累计数据区分。
- 支持房贷、固定支出、家庭消费方向分析。
- 支持生活收支单独清空规划。

## 四、订阅管理

- 支持订阅项目记录。
- 支持订阅金额、周期、到期时间。
- 支持临期提醒。
- 支持已过期、即将到期状态。
- 支持订阅邮件提醒配置。
- 支持 Resend 邮件服务。
- 支持测试邮件。
- 支持扫描并发送订阅提醒邮件。
- 支持订阅成本统计和续费管理基础能力。

## 五、人情往来

- 支持人物关系台账。
- 支持新增人物和新增往来。
- 支持送礼 / 收礼方向。
- 支持事件类型按钮。
- 支持关系类型，例如亲戚、朋友、同事等。
- 支持人物卡片展示差额。
- 支持查看“我差别人 / 别人差我 / 差多少”。
- 支持人物详情弹窗。
- 支持修改往来记录。
- 支持人物年度账。
- 支持年度详情。
- 支持月份趋势。
- 支持事件类型统计。
- 支持导入校验报告规划。
- 支持人物搜索、关系筛选、差额筛选。

## 六、事项清单

- 支持事项新增、编辑、删除。
- 支持待处理、进行中、已完成等状态。
- 支持优先级。
- 支持截止日期。
- 支持今日事项和逾期事项查看。
- 支持事项搜索和筛选。

## 七、笔记收录

- 支持普通笔记。
- 支持碎片想法。
- 支持链接收录。
- 支持 Markdown 内容展示。
- 支持笔记置顶。
- 支持笔记转为事项。
- 支持按类型、分类、标签筛选。

## 八、外部收藏

- 收藏功能定位为“外部资料收录”，不是收藏网站内部内容。
- 支持保存外部链接。
- 支持分类和标签。
- 可作为资料库使用。

## 九、搜索与总览

- 支持全局搜索。
- 支持跨模块搜索事项、账单、笔记、项目等内容。
- 首页总览展示：
  - 待办事项；
  - 收入支出；
  - 订阅提醒；
  - 最近内容；
  - 统计概况。

## 十、设置与数据管理

- 设置面板集中管理数据导入、导出、清空、同步、通知。
- 支持完整 JSON 导出。
- 支持完整 JSON 导入。
- 支持 Excel 财务导入导出。
- 支持服务器数据状态检查。
- 支持通知配置。
- 支持管理员用户管理。

## 十一、部署与安全

- 支持 Docker 部署。
- 支持 VPS 部署。
- 当前 VPS 路径为：

```bash
/opt/New-project
```

- 支持健康检查：

```bash
/healthz
```

- 支持 Docker 数据卷持久化。
- 支持自动备份。
- 支持手动备份：

```bash
npm run backup:data
```

- 支持查看备份：

```bash
npm run backup:list
```

- 支持恢复数据：

```bash
npm run restore:data
```

- 支持生产环境检查：

```bash
npm run check:production
```

- 支持一键部署脚本：

```bash
./scripts/deploy-vps.sh
```

- 支持回滚脚本：

```bash
./scripts/rollback-vps.sh
```

## 当前项目定位

Personal Hub 当前定位为一个个人 / 家庭数据工作台，核心价值是把分散在不同地方的生活数据集中管理：

- 用事项管理日常推进；
- 用生活收支理解家庭现金流；
- 用订阅管理避免自动扣费失控；
- 用人情往来记录关系与差额；
- 用笔记和收藏沉淀碎片信息；
- 用账号、同步、备份和部署能力保证长期可用。
